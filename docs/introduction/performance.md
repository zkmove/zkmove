# Performance

## Overview

zkMove v0.5 delivers significant improvements over v0.4:

- **Proving time** reduced by **0.5×–3×** depending on the workload.
- **Proof size** reduced to a flat **~25 KB**, enabling near-second-level on-chain finality.
